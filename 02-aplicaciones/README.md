# 02 - Aplicaciones

Instalación de todas las aplicaciones esenciales.

> **Nota:** En Homebrew hay dos tipos de instalación:
>
> - `brew install <paquete>` → Herramientas de terminal (node, git, ripgrep)
> - `brew install --cask <app>` → Aplicaciones con interfaz gráfica (iTerm2, Chrome, VSCode)

## Método 1: Instalación individual

### Desde Homebrew (CLI)

```bash

# Utilidades
brew install tree ripgrep fzf lsd
brew install tmux
brew install node
brew install mysql
```

### Desde Homebrew Cask (Apps)

```bash

# Utilidades
brew install --cask itsycal
brew install --cask transmission
brew install --cask balenaetcher
brew install --cask keka
brew install --cask anydesk
```
