## Preferencias del sistema

Configuración recomendada

### Trackpad

- **Apuntar y hacer clic:** Activar
- **Clic con presión secundario:** Activar
- **Clic con un toque:** Activar
- **Gestos:** Configurar según preferencia

### Dock y Barra de Menú

- **Tamaño:** Ajustar al gusto
- **Ampliación:** Activar
- **Minimizar:** Efecto de escala (Genio)
- **Mostrar/ocultar automáticamente:** Según preferencia
- **Posición:** Izquierda, abajo o derecha

### Batería

Ya no existe la opción de máximo nivel de batería.

Solo **activar**,

- "Impedir entrar en reposo automáticamente cuando se apaga la pantalla y se usa el adaptador de corriente"

### Protector de pantalla

- Configurar tiempo de inicio según preferencia
- Hot corners para activar rápidamente

### Finder

- **Mostrar extensiones de archivo:** Activar
- **Mostrar todos los títulos de ventana:** Activar
- **Buscar:** Buscar en el Mac actual (no en la nube)
- **Barra lateral:** Agregar carpetas de uso frecuente
- **Nueva ventana del Finder:** Mostrar carpeta de inicio

### Configurar y Desactivar Fondo de Pantalla

- Quitar el Fondo de Pantalla animado y desactivar el protector de pantalla
