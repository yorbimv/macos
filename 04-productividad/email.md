# Email

Configuración de clientes de correo.

## Apple Mail

### Agregar cuenta

1. **Mail → Agregar cuenta**
2. Seleccionar proveedor (iCloud, Google, Microsoft, etc.)
3. Iniciar sesión y autorizar acceso

### Configuración

**Mail → Ajustes → General:**

- **Comportamiento de bandejas:** Personalizado
- **Fuentes y colores:** Configurar firma
- **Notificaciones:** Activar para bandeja de entrada

## Outlook

### Instalación

1. Descargar desde [office.com](https://office.com)
2. Iniciar sesión con cuenta Microsoft
3. Agregar cuentas adicionales si es necesario

### Configuración

- **Firma:** Preferences → Signatures
- **Notificaciones:** Preferences → Notifications
- **Sync:** Preferences → Sync

## Spark

### Instalación

Descargar desde la App Store.

### Ventajas

- Interfaz limpia y moderna
- Smart Inbox (categoriza emails automáticamente)
- Programar envío de emails
- Snippets de texto

### Configuración

1. Agregar todas las cuentas
2. Configurar Smart Inbox
3. Personalizar temas y atajos

## Comparación

| Cliente | Velocidad | Features | Precio |
