# Cloud Storage

Configuración de servicios de almacenamiento en la nube.

## OneDrive

### Instalación

1. Descargar desde [onedrive.com](https://onedrive.live.com)
2. Iniciar sesión con cuenta Microsoft
3. Elegir carpeta de sync

### Configuración

**Preferencias de OneDrive → Sync y copia de seguridad:**

- **Sync de archivos:** Elegir carpetas
- **Archivos a pedido:** Activar (ahorra espacio local)
- **Copia de seguridad:** Escritorio, Documentos, Imágenes

### Problemas comunes

**Iconos de estado no aparecen en Finder:**

Ver [troubleshooting](../troubleshooting/) → OneDrive

## iCloud Drive

### Configuración

**Ajustes del Sistema → Apple ID → iCloud → iCloud Drive:**

- Activar **Escritorio y Documentos**
- Activar **Optimizar almacenamiento del Mac**

### Compartir con Windows

Si necesitas acceder a archivos de iCloud desde Windows, usar [iCloud para Windows](https://support.apple.com/es-mx/HT204283).

## Google Drive

### Opción 1: Google Drive para Desktop

1. Descargar desde [google.com/drive/download](https://www.google.com/drive/download/)
2. Iniciar sesión
3. Configurar mirror o stream

### Opción 2: Navegador

Acceder desde [drive.google.com](https://drive.google.com)
