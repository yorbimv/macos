# Navegadores

Configuración de Brave y Chrome.

## Brave

### Instalación

```bash
brew install --cask brave-browser
```

### Configuración

**Brave → Ajustes:**

- **Shields:** Activar por defecto
- **Bloqueo de trackers:** Agresivo
- **Motor de búsqueda:** Configurar preferencia
- **Inicio de sesión:** Brave Sync (sin necesidad de email)

### Extensiones recomendadas

- uBlock Origin
- Bitwarden / 1Password
- Dark Reader

## Google Chrome

### Instalación

```bash
brew install --cask google-chrome
```

### Configuración

**Chrome → Ajustes:**

- **Iniciar sesión** con cuenta Google
- **Sync:** Activar marcadores, contraseñas, extensiones
- **Motor de búsqueda:** Google

### Extensiones recomendadas

- uBlock Origin
- Bitwarden / 1Password
- GitHub Copilot
- React Developer Tools
- JSON Viewer

## Compartir marcadores

Si usas múltiples navegadores, exporta/importa marcadores:

1. **Exportar:** Bookmarks → Exportar bookmarks → HTML
2. **Importar:** Bookmarks → Importar bookmarks → HTML
